# Versions
- SimpleRoofPPGNet:
	Ready to use data set for use with the PPGNet.
- SimpleRoofTXT:
	General format for any usage. *.txt*-Files are structured as following:
	- Junctions (#1#)
	- Line segments (#2#)
	- Adjacency matrix (#3#)
