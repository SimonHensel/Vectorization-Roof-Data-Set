# Vectorzation-Roof-Data-Set
A Roof Data set for training and testing the PPGNet.

**Versions:**
- SimpleRoofPPGNet:
	Ready to use data set for use with the PPGNet.
- SimpleRoofTXT:
	General format for any usage. *.txt*-Files are structured as following:
	- Junctions
	- Line segments
	- Adjacency matrix
